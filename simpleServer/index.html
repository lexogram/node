A Simple Node.js Web Server

This page is served by a very simple node.js server.

This page contains:

  • Formatted HTML text
  • JPEG and GIF images
  • A CSS layout
  • Interactive JavaScript from an external file (move the mouse over the images)
  • A link to another page
  • A broken link to a page that is missing
... and most importantly:
  • A link to a tutorial that allows you to build this node.js site for yourself.
